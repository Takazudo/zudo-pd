# ICL7660 接続図（超分かりやすい版）

## CAP+とCAP-の正しい接続

### 接続の基本ルール
**フライングキャパシタ（C9）はPIN1とPIN4の間に直接接続**

```
      フライングキャパシタ C9 (10µF)
           ┌─────────┐
PIN1 ──────┤+       -├────── PIN4
(CAP+)     └─────────┘     (CAP-)
```

### 詳細回路図
```
+15V ─────┬─────────────────────────────────────────┬─→ +15V (他回路へ)
          │                                         │
          │           ICL7660M/TR                   │
          │         ┌──────────────┐                │
          └─────────┤2,8  V+   CAP+│1──┐            │
                    │              │   │            │
           GND ─────┤3   GND   CAP-│4──┼─┐          │
                    │              │   │ │          │
          OPEN ─────┤6   LV    V-  │5──┼─┼──────────┴─→ -15V
                    │              │   │ │
          OPEN ─────┤7   OSC       │   │ │
                    └──────────────┘   │ │
                                       │ │
                     C9: 10µF セラミック │ │
                    ┌───────────────────┘ │
                    │                     │
                    └─────────────────────┘
                    
                    C10: 10µF セラミック
                    ┌─────────────────────────┐
                    │                         │
            -15V ───┼─────────────────────────┼─── GND
                    │                         │
                    └─────────────────────────┘
```

## 接続を段階的に説明

### ステップ1: 電源接続
```
+15V → PIN2, PIN8 (V+)
GND  → PIN3 (GND)
```

### ステップ2: フライングキャパシタ接続
```
PIN1 (CAP+) ──┐
              │ C9 (10µF セラミック)
PIN4 (CAP-) ──┘
```
**重要**: C9はPIN1とPIN4を直接つなぐだけ！他には何も接続しない

### ステップ3: 出力フィルタ接続
```
PIN5 (V-) ──┐
            │ C10 (10µF セラミック)  
GND ────────┘
```

### ステップ4: 制御ピン
```
PIN6 (LV)  → 何も接続しない（OPEN）
PIN7 (OSC) → 何も接続しない（OPEN）
```

## 実際の基板配線

### パターン配線例
```
           +15V供給ライン
              │
         ┌────┴────┐
    PIN1 │         │ PIN8,2
  ┌──────┤         ├──────┐
  │ C9   │ ICL7660 │      │ +15V
  │ 10µF │         │      │
  └──────┤         ├──────┘
    PIN4 │         │ PIN3
         └────┬────┘
              │
             GND
              │
         ┌────┴────┐
         │ C10 10µF│
    PIN5 ├─────────┤ GND
         │  -15V   │
         └─────────┘
```

### 部品配置のコツ
1. **C9 (フライング)**: ICL7660のすぐ近くに配置
2. **C10 (出力)**: PIN5からGNDへ短い配線
3. **GNDプレーン**: しっかりしたGND面を確保
4. **+15Vライン**: PIN2とPIN8両方に確実に供給

## よくある間違い

### ❌ 間違った接続
```
+15V → PIN1  (これは間違い！PIN1はCAP+)
C9の片方をGNDに接続  (これも間違い！)
```

### ✅ 正しい接続
```
+15V → PIN2, PIN8 のみ
C9 → PIN1とPIN4の間のみ
```

## 動作確認方法

### 電圧測定ポイント
1. **PIN2,8**: +15V確認
2. **PIN3**: 0V (GND) 確認  
3. **PIN5**: -14V～-15V確認
4. **PIN1,4間**: 動作中は変動（測定困難）

### 正常動作の目安
- **無負荷時**: -14.8V～-15V
- **10mA負荷**: -14.5V程度
- **30mA負荷**: -14V程度
