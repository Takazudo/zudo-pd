```
+15V ─────┬──────────────────────────────────────┬─→ +15V (他回路へ)
          │                                      │
          │            ICL7660M/TR               │
          │          ┌──────────────┐            │
          └──────────┤8,2  V+   CAP+│1───┬──────┤
                     │              │    │      │
                     │3    GND  CAP-│4───┼──────┼── C9: 10µF/25V
                     │              │    │      │   (フライング)
                     │7    OSC   V- │5───┼──────┴─────────────┬─→ -15V
                     │              │    │                    │
                     │6    LV       │    │                    │

                           │             │                    │
```

### ピン配置詳細 (SOP-8)
```
      1   2   3   4
    ┌───────────────┐
  8 │○              │ 5
    │   ICL7660M    │
  7 │               │ 6
    └───────────────┘

PIN1: CAP+ (フライングキャパシタ+)
PIN2: V+   (正電源入力)
PIN3: GND  (グランド)
PIN4: CAP- (フライングキャパシタ-)
PIN5: V-   (負電圧出力)
PIN6: LV   (低電圧モード選択、OPEN)
PIN7: OSC  (発振器制御、OPEN)
PIN8: V+   (正電源入力、PIN2と接続)
```

### 接続リスト

| 接続元 | 接続先 | 部品 | 説明 |
|--------|--------|------|------|
| +15V | PIN2,8 | - | 正電源入力 |
| PIN1 | C9の+ | 10µF | フライングキャパシタ正極 |
| PIN4 | C9の- | 10µF | フライングキャパシタ負極 |
| PIN3 | GND | - | グランド |
| PIN5 | C10の+ | 10µF | 出力フィルタ |
| C10の- | GND | - | 出力フィルタグランド |
| PIN6 | OPEN | - | 通常動作モード |
| PIN7 | OPEN | - | 内蔵発振器使用 |
| PIN5 | -15V出力 | - | 負電圧出力 |

### 動作説明

#### フェーズ1（充電）
```
+15V → PIN2,8 → 内部スイッチ → C9(10µF) → GND
```
- C9が+15Vに充電される

#### フェーズ2（放電・反転）
```
GND → C9(10µF) → 内部スイッチ → PIN5 → C10 → GND
```
- C9の電荷が反転してPIN5から-15V出力

### 部品仕様（JLCPCB選定済み）

#### ICL7660M/TR
- **部品番号**: C356724
- **入力**: +15V
- **出力**: -15V (理論値), -14V～-14.5V (実用値)
- **最大電流**: 45mA (軽負荷時)

#### フライングキャパシタ (C9)
- **部品**: CL21A106KAYNNNE (C15850)
- **容量**: 10µF 25V X5R
- **パッケージ**: 0805
- **価格**: $0.0101

#### 出力フィルタキャパシタ (C10)
- **部品**: CL21A106KAYNNNE (C15850)
- **容量**: 10µF 25V X5R  
- **パッケージ**: 0805
- **価格**: $0.0101

### 重要な設計ポイント

1. **PIN2とPIN8の接続**: 両方とも+15Vに接続必要
2. **C9の極性**: セラミックなので極性なし、接続方向自由
3. **C10の配置**: PIN5からGNDへ、ノイズフィルタとして重要
4. **PIN6,7はOPEN**: 通常動作・内蔵発振器使用
5. **出力電流制限**: 45mA程度、後段LM2596S必須

### システム全体での位置
```
USB-C PD → CH224Q → +15V ┬→ LM2596S #1 → +13.5V
                         ├→ LM2596S #2 → +7.5V  
                         └→ ICL7660 → -15V → LM2596S #3 → -13.5V
```

この修正版が正しいICL7660の接続方法です。
