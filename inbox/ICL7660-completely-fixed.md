# ICL7660 正しい回路図（完全修正版）

## 完全に分離された+15Vと-15V

```
【入力】+15V電源ライン
+15V ─────┬─→ 他のDC-DCコンバータへ
          │
          │    ICL7660M/TR (SOP-8)
          │  ┌──────────────┐
          └──┤2,8  V+   CAP+│1──┐
             │              │   │
    GND ─────┤3   GND   CAP-│4──┼─┐
             │              │   │ │
    OPEN ────┤6   LV    V-  │5──┼─┼───┐
             │              │   │ │   │
    OPEN ────┤7   OSC       │   │ │   │
             └──────────────┘   │ │   │
                                │ │   │
              C9: 10µF          │ │   │
             ┌──────────────────┘ │   │

             └────────────────────┘   │
                                      │
              C10: 10µF               │
             ┌────────────────────────┘
             │
             ├─→ -15V 【出力】負電源ライン
             │
            GND
```

## 接続リスト（明確版）

### 電源接続
| 端子 | 接続先 | 説明 |
|------|--------|------|
| +15V | PIN2, PIN8 | 正電源入力 |
| GND | PIN3 | グランド |

### コンデンサ接続
| 部品 | 接続 | 機能 |
|------|------|------|
| C9 (10µF) | PIN1 ↔ PIN4 | フライングキャパシタ |
| C10 (10µF) | PIN5 ↔ GND | 出力フィルタ |

### 制御ピン
| ピン | 接続 |
|------|------|
| PIN6 (LV) | 開放 |
| PIN7 (OSC) | 開放 |

### 出力
| 端子 | 電圧 |
|------|------|
| PIN5 | -15V |

## システム内での位置

```
USB-C → CH224Q → +15V ┬→ LM2596S #1 → +13.5V
                      │
                      ├→ LM2596S #2 → +7.5V
                      │
                      └→ ICL7660

                         ICL7660 → -15V → LM2596S #3 → -13.5V

                    【共通GND】
```

## 重要ポイント

1. **+15Vと-15Vは物理的に分離**
   - +15V: 入力電源ライン
   - -15V: 出力電源ライン
   - 2つの線は繋がっていない

2. **共通点はGNDのみ**
   - 全回路で共通のグランド
   - 0Vの基準点

3. **ICL7660は変換器**
   - +15V（入力） → -15V（出力）
   - 内部で電圧を反転

4. **フライングキャパシタC9**
   - PIN1とPIN4の間のみ
   - 他には何も接続しない
