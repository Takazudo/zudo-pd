## オリジナルのアイデア

信頼性の高いDoepferの電源では、そのようにDC電圧を取得しつつも、最終的にはリニアレギュレーター（LM7812やLM7912のような）とキャパシタにより、スイッチングノイズを提言させる実装をしている。

これと同じように、USB-PDから電源を取得し、そしてモジュラーシンセサイザーに必要な

+12V
-12V
+5V

の電圧を作りたい。
そしてこの時の電圧は以下ほどが必要（正確にこの値である必要は無い）

+12V: 1200mA
-12V: 800mA
+5V: 500mA

この値は、それぞれの電圧が概ねこのくらいの割合でモジュラーシンセサイザーでは利用されるためである。
このようなモジュラーシンセ用電源を実装。
