### Feedback Network (Voltage Divider)

```
┌──────────────┐     ┌─────────┐     ┌──────────┐
│ +13.5V Out   │     │   R1    │     │    R2    │
│              ├─────┤  10kΩ   ├──┬──┤   1kΩ    ├─→ GND
└──────────────┘     └─────────┘  │  └──────────┘
                                  │
                ┌──────────────────┐
                │   U2 (LM2596S)   │
                │2. FB             │
                └──────────────────┘

Tap voltage = +13.5V × R2/(R1+R2) = 13.5V × 1kΩ/11kΩ = 1.23V
U2 maintains FB pin at 1.23V reference by adjusting duty cycle
```

**Key Points:**
- **D1 cathode** connects to switching node (between U2 VOUT and L1)
- **D1 anode** connects to GND
- **C3** is in parallel (output → C3 → GND), NOT in series
- **ON pin 3** tied to VIN for always-on operation
- **FB pin 2** connects to tap point (see feedback diagram)
