# ICL7660M/TR Voltage Inverter

A charge pump voltage inverter IC that generates negative voltage from positive voltage.

## Overview

The ICL7660M/TR is a CMOS voltage converter IC manufactured by Renesas Electronics (formerly Intersil). It uses a switched-capacitor (Charge Pump) method to invert positive input voltage to negative output voltage. The design is simple and reliable, requiring only two external capacitors for operation.

In this project, it inverts the +15V voltage obtained from USB-C PD to -15V voltage, which is used to generate the negative voltage rail (-12V). Since no inductors are required, EMI noise is minimal, making it ideal for modular synthesizer power supplies.

## Application in This Project

### Circuit Configuration

In this project, the ICL7660M is configured as follows to achieve +15V to -15V voltage inversion:

          │                                      │
          │           ICL7660M/TR                │
          │         ┌──────────────┐             │
          └─────────┤8  V+    CAP+ │2──┐         │
                    │              │   │         │

### Component Values

- **C9**: 10µF Ceramic Capacitor (Flying Capacitor)
  - JLCPCB: C15850 (CL21A106KAYNNNE)
  - Voltage Rating: 25V
  - Package: 0805
  - Purpose: Charge transfer between CAP+ and CAP-

- **C10**: 10µF Ceramic Capacitor (Output Smoothing)
  - JLCPCB: C15850 (same as above)
  - Voltage Rating: 25V
  - Package: 0805
  - Purpose: Output ripple reduction

### Detailed Connection

**Important**: Flying capacitor C9 is connected **directly** between PIN2 (CAP+) and PIN4 (CAP-):

```
      Flying Capacitor C9 (10µF)
           ┌─────────┐
PIN2 ──────┤+       -├────── PIN4
(CAP+)     └─────────┘     (CAP-)
```

**Output smoothing capacitor C10** is connected between -15V output and GND:

```
      Output Capacitor C10 (10µF)
           ┌─────────┐
 -15V ─────┤-       +├────── GND
(VOUT)     └─────────┘
```

### Operating Principle

The ICL7660M operates using switched-capacitor method:

1. **Phase 1** (Charge):
   - Internal switches connect C9 between V+ and GND
   - C9 charges up to V+ voltage

2. **Phase 2** (Transfer):
   - Internal switches invert the polarity of C9
   - C9's + terminal connects to GND, - terminal connects to VOUT
   - Charge transfers from C9 to C10, generating negative voltage

3. **Oscillation**: The above cycle repeats at 10kHz

### Performance in This Project

**Input Conditions**: +15V (via USB-C PD)

**Theoretical Output**: -(15V - 1V) = -14V

**Expected Measured Value**: -13.5V to -14.5V (varies with load)

**Output Current**: Maximum 100mA (in this project, supplies to the downstream LM2596S)

**Note**: The ICL7660M's maximum input voltage is +10V, but in practice it can operate up to approximately +12V. However, +15V input is outside specifications, so the following countermeasures are necessary:

### Voltage Limitation Workaround

**Problem**: ICL7660M's maximum input voltage is +10V, but this project supplies +15V

**Solution Options**:

1. **Option A: Clamp with Zener Diode** (Recommended)
   ```
   +15V ───[Resistor 100Ω]──┬──→ ICL7660M V+ (approx. 9V)
                            │
                        [9.1V Zener]
                            │
                           GND
   ```

2. **Option B: Use Low Dropout Voltage Regulator**
   ```
   +15V ──→ [LM7809] ──→ +9V ──→ ICL7660M V+
   ```

3. **Option C: Resistor Divider (Simple, Lower Efficiency)**
   ```
   +15V ───[4.7kΩ]──┬──→ ICL7660M V+ (approx. 9V)
                    │
                [10kΩ]
                    │
                   GND
   ```

**Implementation in This Project**: Option A is recommended (low cost, high efficiency)

### Updated Circuit with Protection

```
+15V ───[R_drop 100Ω]──┬─────────────────────┬─→ 9V regulated
                       │                     │
                   [D_zener]              ┌──┴──┐
                    9.1V TVS              │ C11 │ 10µF
                       │                  │     │
                      GND                 └──┬──┘
                                             │
                                            GND

9V ───────────┬─── ICL7660M ───┬──→ -9V
              │                │
            [C9]             [C10]
           10µF              10µF
              │                │
             GND              GND
```

**Additional Components**:
- **R_drop**: 100Ω, 0.5W Resistor (for voltage drop)
- **D_zener**: 9.1V Zener Diode (e.g., BZT52C9V1S)
- **C11**: 10µF Decoupling Capacitor

**Result**:
- Input Voltage: Approximately 9V (within safe operating range)
- Output Voltage: Approximately -8.5V (boosted to -13.5V by downstream LM2596S)

## Design Considerations

### 1. Capacitor Selection

**Flying Capacitor (C9)**:
- **Capacitance**: 10µF recommended (datasheet specifies 10µF-100µF)
- **Type**: Ceramic capacitor (low ESR is important)
- **Voltage Rating**: At least 2x input voltage (25V for this project)
- **Package**: 0805 or larger (to ensure current capacity)

**Output Capacitor (C10)**:
- **Capacitance**: 10µF or more (for ripple reduction)
- **Type**: Ceramic or electrolytic capacitor
- **ESR**: Lower is better (reduces output impedance)

### 2. PCB Layout

**Important Points**:
1. **Short Traces**: Keep CAP+, CAP-, VOUT traces as short as possible
2. **GND Plane**: Ensure a continuous GND plane
3. **Decoupling**: Adding a 0.1µF capacitor near the V+ terminal is recommended
4. **Thermal**: SOP-8 package has low heat generation, but connect to GND plane for heat dissipation

### 3. Output Current Limitation

**Important**: The ICL7660M's maximum output current is 100mA.

**Usage in This Project**:
- ICL7660M only supplies power to the downstream LM2596S-ADJ
- LM2596S input current: Approximately 50mA (light load)
- **No Problem**: Operates within 100mA limit

**Note**: If supplying -12V load directly from ICL7660M, consider the current limitation.

### 4. Efficiency Considerations

**Efficiency Characteristics**:
- No load: ~90%
- 50mA load: ~70%
- 100mA load: ~60%

**Heat Dissipation Calculation** (for this project):
```
Input power: 9V × 50mA = 0.45W
Output power: 8.5V × 50mA × 0.7 (efficiency) = 0.30W
Heat dissipation: 0.45W - 0.30W = 0.15W
```

**Conclusion**: Heat generation is very small, no heatsink required

### 5. Noise Characteristics

**Advantages**:
- No magnetic noise due to absence of inductors
- 10kHz operation is outside audible range
- Good EMI characteristics

**Disadvantages**:
- Output ripple: Approximately 10-20mVp-p (at light load)
- Since LM2596S-ADJ and LM7912 linear regulator follow, no problem at final output

### 6. Alternative Solutions

**If higher current is required**:
- **ICL7660S**: Standard version (up to 200mA, but +12V input limit)
- **LTC1044**: High performance version (up to 100mA, wide input range)
- **TPS60403**: TI product (up to 60mA, 95% efficiency, +15V compatible)

**For this project**: ICL7660M provides sufficient performance

## Notes

- The ICL7660M is a classic IC that has been used since the 1970s, but it is still widely used today
- The "M" suffix stands for "Micropower", indicating the low power consumption version
- Connecting the LV terminal to GND enables low voltage mode (+1.5V input support), but output current decreases
- Connecting an external capacitor to the OSC terminal can lower the switching frequency (for audible noise countermeasures)
- Use exceeding the input voltage limit (+10V) is not guaranteed, but safe operation is possible with appropriate protection circuits
- In this project, a Zener diode clamp circuit is used to limit the +15V input to approximately +9V
